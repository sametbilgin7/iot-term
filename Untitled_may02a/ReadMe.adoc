= Project: IoT Solar Tracker Panel System

This project implements an ESP32-based solar tracker panel system with real-time energy monitoring and IoT dashboard integration.
Two LDR sensors detect the stronger light direction, and a servo motor rotates the panel toward that direction.
An INA219 sensor measures voltage and current so the system can calculate power output.
Live values and system status are shown on a 128x64 OLED display and also sent over Wi-Fi to Arduino IoT Cloud.

NOTE: In the original term proposal, a buzzer was planned as an extra feature. The implemented project does not include the buzzer due to hardware availability, and all other planned core features are implemented.

== Step 1: Installation
1. Install Arduino IDE 2.x.
2. Install ESP32 board package in Arduino IDE.
3. Install required libraries:
  - `ArduinoIoTCloud`
  - `Arduino_ConnectionHandler`
  - `Adafruit INA219`
  - `Adafruit GFX`
  - `Adafruit SSD1306`
  - `ESP32Servo`
4. Open `Untitled_may02a.ino`.
5. Update Wi-Fi and Arduino Cloud credentials in `arduino_secrets.h` and `thingProperties.h`.
6. Select the correct ESP32 board and port, then upload.

== Step 2: Assemble the circuit

Connect components according to your circuit layout:

1. LDR-Left and LDR-Right to ESP32 analog input pins.
2. Servo motor signal pin to ESP32 PWM-capable digital pin.
3. INA219 via I2C (`SDA`/`SCL`).
4. OLED 128x64 via I2C (`SDA`/`SCL`).
5. Common GND for all modules and stable power rails.

== Step 3: Load the code

Upload `Untitled_may02a.ino` to ESP32 and open Serial Monitor for diagnostics.
After boot, verify:

1. LDR readings change with light direction.
2. Servo tracks stronger light source.
3. INA219 values (V, A) update.
4. OLED shows tracking + energy status.
5. Cloud dashboard receives telemetry and control states.
